# Raizes do Equilibrio

Projeto autoral desenvolvido para a subcategoria 3 do Concurso Agrinho 2026, com base no tema:

`Agro forte, futuro sustentavel: equilibrio entre producao e meio ambiente`

## Conceito

O site apresenta uma narrativa visual sobre a relacao entre campo e cidade, destacando como
tecnologia, manejo responsavel, uso inteligente da agua e consumo consciente podem fortalecer o agro
sem romper o equilibrio ambiental.

## Tecnologias

- HTML semantico
- CSS responsivo
- JavaScript puro, com manipulacao de DOM
- Service Worker simples para cache da experiencia

## Interacoes implementadas

- Simulador de equilibrio com leitura dinamica de cenario
- Mensagens contextuais por horario
- Alternancia de paisagem
- Gerador de manifesto personalizado
- Mural local com persistencia no navegador

## Estrutura

- `index.html`: estrutura principal do site
- `styles.css`: identidade visual, animacoes e responsividade
- `js/main.js`: interacoes, simulador, manifesto e persistencia local
- `assets/brand-icon.svg`: icone do projeto
- `manifest.webmanifest`: configuracao do app web
- `sw.js`: cache offline basico

## Publicacao

O projeto foi preparado para publicacao estatica em GitHub Pages ou Vercel.

Checklist recomendado para a inscricao:

1. Deixar o repositorio publico no GitHub.
2. Preencher o campo `About` do repositorio com a descricao do projeto.
3. Adicionar o link publicado do GitHub Pages ou Vercel no campo `Website`.
4. Adicionar o topic oficial `agrinho` nos topics do repositorio.
5. Conferir se `index.html`, `styles.css` e `js/main.js` estao carregando sem erros.

## Observacao

O projeto foi mantido sem frameworks e sem CSS/JavaScript inline nos arquivos HTML, conforme o regulamento da subcategoria 3.

## Registro de apoio por IA

Para manter transparencia com a rubrica do concurso, este projeto considera apoio de IA na criacao da base visual,
do icone SVG, da estrutura HTML e dos textos iniciais.

Descricao do prompt utilizado como base:

`Criar um site autoral em HTML, CSS e JavaScript puro para o Agrinho 2026, com o tema Agro forte, futuro sustentavel: equilibrio entre producao e meio ambiente, sem frameworks, com layout responsivo, interacoes no DOM, identidade visual propria e foco em pontuacao maxima na rubrica da subcategoria 3.`
